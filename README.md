# Task 6: Sales Trend Analysis

## Objective
Analyze monthly revenue and order volume using SQL aggregation.

## Tools Used
- SQLite Online Compiler

## Steps Performed
- Created online_sales table
- Inserted sample data
- Used GROUP BY for monthly analysis
- Calculated revenue using SUM()
- Counted orders using COUNT(DISTINCT)
  
## Insights
- Sales show a consistent upward trend from January to April.
- April recorded the highest revenue, while January had the lowest.
- A significant increase in revenue is observed from February to March.
- Order volume remains constant, indicating higher revenue is driven by increased order value rather than more orders.
